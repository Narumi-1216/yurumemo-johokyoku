# 【AI BGM】20本投稿して登録者5人 → 3ヶ月半で収益化

## 数字で見る経過

| 時点 | 状態 |
|---|---|
| 2025年12月 | 開始（息子の一言がきっかけ。息子は5本でリタイア） |
| 20本投稿時点 | 登録者5人（うち3人は家族） |
| 投稿約1ヶ月後 | 1本の動画が突然伸びて1万回再生、チャンネル全体を牽引 |
| 登録者300人頃 | 尊敬する先輩にDM →「結果を出すしかない」環境を自作 |
| 開始から3ヶ月半 | 登録者1,000人達成、収益化 |

## 使用ツール

- 曲づくり → Suno（最下位プランから4,500円プランへ早期変更。試行回数が重要）
- 画像生成 → Midjourney
- サムネ → Canva
- 動画編集 → Filmora

## 継続できた要因（本人の分析）

- 最初の20本は「作業に慣れる準備期間」であり無駄ではなかった
- オフ会で「行動する人の空気」に背中を押された（最初はそんなもん、という優しい現実）
- 2週間チャレンジ：過去2回失敗 → 「毎日触れる（触れなければいけない）環境」で継続成功
- 継続の理由は「やる気」ではなく「環境」
- 質より量、下手な鉄砲も数撃ちゃ当たる精神
- 「当たる動画を作る」より「続けたから当たりに出会えた」

## 教訓

- 100人を超えた頃から数字の伸び方が変わる（1日1人 → 5人 → 10人…）
- 一人で頑張るより「誰かに見てもらう環境」を作る
- 「もう遅いかどうか」ではなく「まずやってみる」
